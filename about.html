About This Project

Your personal study companion, powered by AI

💡 What is this?

Académie is a comprehensive, AI-powered learning platform designed to help students master their coursework through adaptive learning techniques, gamification, and spaced repetition.

Currently featuring Biology Quiz 2, with plans to expand to Chemistry, Mathematics, and more subjects in the future.

✨ Key Features

🎴
Smart Flashcards
Spaced repetition algorithm adapts to your learning
📝
Practice Tests
Adaptive feedback identifies weak areas
🎮
Learning Games
Matching, speed challenges, and more
📊
Progress Tracking
Detailed analytics and insights

React Tailwind CSS LocalStorage API HTML5 JavaScript GitHub Pages Anthropic Google Firebase
